University of California, Berkeley, JavaScript Subset Compiler (C++)

  • built a compiler for JSS which I adopted from JavaScript and is a subset of it. Besides traditional control-flow (if, switch) and iteration (while, for) statements, JSS has many traits of a modern programming language, such as closure, dynamic type, class, exception handling, lambda function and garbage collection.
  • This compiler is a part of a program synthesis project which aims at automatically constructing a program that satisfies a given set of desired behaviors of users' intent expressed in the form of a natural language. It mainly includes four modules:
    • Lexer: tokenize JSS source code.
    • Syntatic Parser: implement context-free grammar, finite state machine, recursive descending parser, regular expression, and build up the abstract syntax tree(AST) for JSS source
    • Semantic Analyzer: compile JSS tokens to JSS VM code.
    • Virtual Machine: to run compiled JSS code.
